# 🌦️ Weather Dashboard

A responsive Weather Dashboard built using **HTML, CSS, and JavaScript** that allows users to search for any city and view real-time weather information using a weather API.

## 🚀 Live Demo

🔗 https://github.com/Srinivasvasam45/Weather_Report.git

## 📌 Features

* Search weather by city name
* Display current temperature
* Show weather conditions
* Display humidity and wind speed
* Dynamic weather updates using API data
* Responsive design for desktop and mobile devices
* User-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Weather API

## ⚙️ How It Works

1. User enters a city name.
2. The application sends a request to the weather API.
3. Weather data is fetched asynchronously.
4. The UI updates dynamically with the latest weather information.
5. Error messages are displayed for invalid city names or failed API requests.
